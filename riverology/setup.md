# Install & Setup

## A. Install the plugin

Install from Epic Launcher/FAB, or copy into:
```
MyProject/Plugins/
```
Enable **Riverology** in **Edit → Plugins** and **Restart**.

## B. Project settings

No special renderer settings are required for Riverology alone.  
If you also use **Oceanology** in the same project, apply the **RendererSettings** from the corresponding Oceanology guide and keep Riverology on defaults.

## C. Verify

1. Create or open a test level.
2. Add a river spline from the Riverology toolset.
3. Simulate and confirm flow, foam, and buoyancy work as expected.
