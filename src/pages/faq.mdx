# Frequently Asked Questions (FAQ)

A few common questions. If you can't find yours, drop by <a href="https://discord.gg/VHJGBDR2as" target="_blank" rel="noopener">Discord</a>.

## What Unreal Engine versions are supported?
We target UE5.x. Check release notes on the blog for exact compatibility per release.

## Does NextGen replace Legacy?
No. **Legacy** is maintained; **NextGen** introduces new architectures and optimizations.

## How do I report a bug or request a feature?
Use the **Community** page links or open a thread in #support on Discord.
